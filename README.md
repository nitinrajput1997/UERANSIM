# UERANSIM

### Install UERANSIM
```bash
# install cmake and other packages
sudo apt update
sudo apt upgrade
sudo apt install iproute2
sudo snap install cmake --classic
sudo apt install gcc
sudo apt install g++
sudo apt install libsctp-dev
```

```bash
# clone ueransim
git clone https://github.com/aligungr/UERANSIM
cd UERANSIM
make
```
